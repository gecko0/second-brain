# Inbox

Unprocessed captures go here.

Use this folder for quick notes, copied links, rough ideas, meeting fragments, and anything that does not yet have a clear home.

## Processing Checklist

- Is this actionable? Move it to `10-projects/`.
- Is this an ongoing responsibility? Move it to `20-areas/`.
- Is this reference material? Move it to `30-resources/`.
- Is this a durable idea? Distill it into `40-notes/`.
- Is this no longer active but still useful? Move it to `50-archive/`.
