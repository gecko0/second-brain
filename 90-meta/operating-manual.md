# Operating Manual

## Daily Use

1. Capture quickly into `00-inbox/` or `TODAY.md`.
2. Keep `TODAY.md` focused on current work.
3. Move durable or reusable knowledge out of the inbox.

## Weekly Review

1. Empty or reduce `00-inbox/`.
2. Review active projects in `10-projects/`.
3. Update `TODAY.md`.
4. Archive stale project material.
5. Promote reusable insights into `40-notes/`.

## AI Agent Workflow

When asking an AI agent to add knowledge:

- Provide the source, context, or goal.
- Ask it to place the note in the correct folder.
- Ask it to update indexes when new important topics are added.

When asking an AI agent questions:

- Ask it to cite the note paths it used.
- Ask it to distinguish vault evidence from inference.
