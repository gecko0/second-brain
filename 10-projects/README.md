# Projects

Projects are active outcomes with a finish line.

Projects are grouped by life context first, then by project or project group.

Use:

- [Personal](personal/README.md) for personal outcomes.
- [Euristiq](euristiq/README.md) for company work, including internal initiatives and client projects.

Each project should have its own folder and a `README.md` describing the outcome, status, next actions, and relevant links.
