# Euristiq Client Projects

Client work is organized by client first, then by project.

Use this shape:

```text
clients/
  client-name/
    README.md
    project-name/
      README.md
      decisions.md
      notes.md
      meetings.md
      risks.md
```

The client `README.md` stores account-level context that can survive across multiple projects. Individual project folders store delivery work with a finish line.
