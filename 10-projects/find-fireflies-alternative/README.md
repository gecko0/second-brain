# Find Fireflies Alternative

## Outcome

Choose a replacement for Fireflies that fits my meeting transcription, search, summarization, privacy, workflow, and cost requirements.

The main decision driver is price. Fireflies is not cheap, and I have many meetings each day. I do not need every meeting transcribed, but I do need meetings recorded reliably.

## Status

- State: exploring
- Started: 2026-05-16

## Candidate Types

- Cloud SaaS
- Open-source local
- Self-hosted
- Hybrid workflows

## Next Actions

- Define must-have requirements.
- Identify cheap or free ways to record many meetings.
- Build candidate list.
- Compare pricing, recording capacity, privacy, transcription quality, and workflow fit.
- Test top candidates against the same sample meeting.
- Decide whether to migrate from Fireflies.
