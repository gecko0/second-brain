# Terminal UI Frameworks

## Summary

Reference links for building rich terminal user interfaces.

## Key Points

- [OpenTUI](https://opentui.com/) is a native terminal UI core written in Zig with TypeScript bindings, first-class React and Solid support, and a C ABI for use from other languages.
- It is useful when a CLI needs a richer interactive interface than plain prompts or command output.
- Notable capabilities include component-based layout, Yoga-powered flexbox sizing, keyboard focus/input handling, tree-sitter syntax highlighting, built-in components, and terminal animations.
