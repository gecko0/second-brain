# Sim AI Agent Platform

## Summary

Sim is an open-source platform for building, deploying, and orchestrating AI agents and agentic workflows.

## Key Points

- [simstudioai/sim](https://github.com/simstudioai/sim) provides a visual workflow builder for connecting agents, tools, integrations, and blocks on a canvas.
- It positions itself as a central orchestration layer for an AI workforce, with support for 1,000+ integrations and multiple LLM providers.
- Useful areas to inspect: visual agent workflow UX, integration architecture, RAG/vector database support, copilot-assisted workflow editing, self-hosting model, and background job/runtime patterns.
- Self-hosting options include `npx simstudio`, Docker Compose, and manual setup with Bun, Node.js 20+, PostgreSQL, and pgvector.
- Tech stack includes Next.js App Router, Bun, PostgreSQL, Drizzle ORM, Better Auth, Zod, shadcn/ui, Tailwind CSS, Zustand, TanStack Query, React Flow, Socket.io, Trigger.dev, E2B, and isolated-vm.
