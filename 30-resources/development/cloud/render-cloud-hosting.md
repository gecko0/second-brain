# Render Cloud Hosting

## Summary

Render is a cloud hosting platform for deploying and operating web apps, APIs, workers, databases, static sites, cron jobs, and related infrastructure without managing servers directly.

## Key Points

- [Render](https://render.com/) is a platform-as-a-service style hosting provider for shipping applications from a repo or Docker image.
- Core service types include static sites, web services, private services, background workers, cron jobs, managed Postgres, and Redis-compatible key-value storage.
- Platform features include automatic deploys, preview environments, autoscaling, private networking, persistent disks, zero-downtime deploys, rollbacks, logs, monitoring, TLS, and infrastructure as code through `render.yaml`.
- Useful mental model: Render overlaps with Heroku, Railway, Fly.io, and some Vercel backend/full-stack use cases, with more built-in infrastructure primitives than static frontend hosting alone.
