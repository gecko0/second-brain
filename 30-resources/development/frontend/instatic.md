# Instatic

Source: https://github.com/corebunch/instatic
Captured: 2026-06-28

Instatic is a self-hosted visual CMS from CoreBunch. It combines a visual editor, content engine, media, auth, forms, plugins, and publishing in one Bun server.

Key points:

- MIT licensed.
- Runs as a self-hosted CMS.
- Uses Bun, TypeScript, and either SQLite or Postgres.
- Publishes clean semantic HTML and compact CSS, aiming for static-file-like performance.
- Includes visual page building, reusable components, templates, loops, forms, media management, role-based access, audit logs, and a plugin system.
- The plugin system runs plugins inside a QuickJS-WASM sandbox with explicit permissions.

Potential use: evaluate as a simpler self-hosted alternative to a headless CMS plus site builder plus hosting stack.
