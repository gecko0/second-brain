# AI SDK Tools

## Summary

AI SDK Tools is a toolkit that extends the Vercel AI SDK for building more advanced AI applications beyond simple chat interfaces.

## Key Points

- [AI SDK Tools](https://ai-sdk-tools.dev/) provides utilities for multi-agent orchestration, state management, debugging, structured artifact streaming, caching, and persistent memory.
- The unified npm package is `ai-sdk-tools`; individual packages include `@ai-sdk-tools/store`, `@ai-sdk-tools/devtools`, `@ai-sdk-tools/artifacts`, `@ai-sdk-tools/agents`, `@ai-sdk-tools/cache`, and `@ai-sdk-tools/memory`.
- Useful areas to inspect: Zustand-based AI chat state, in-app devtools for tool calls and execution flow, type-safe artifact streaming to React components, automatic agent handoffs/routing, and caching for expensive tool executions.
- The GitHub README notes the package is in active development and recommends pinning versions in production because breaking changes may occur.
