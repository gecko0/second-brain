# Spokenly AI Dictation

## Summary

Spokenly is an AI dictation app for Mac, iPhone, and Windows that provides system-wide speech-to-text with local and cloud model options.

## User Context

- Captured on 2026-06-08 as a good-looking option for a local version of a WisperFlow-style voice dictation workflow.

## Key Points

- [Spokenly](https://spokenly.app/) lets users press a shortcut, speak, and insert text at the cursor in any app that accepts text input.
- Local models are free and unlimited, using Whisper and Parakeet with offline support.
- Bring-your-own-key mode supports providers such as OpenAI, Deepgram, and Groq for real-time transcription.
- Spokenly Pro includes cloud models without requiring personal API keys.
- Privacy angle: local-only mode blocks network requests so voice data stays on-device with local models.
- Supports 100+ languages with automatic language detection, though quality varies by model and language.
- Additional features include AI cleanup/formatting prompts, dictation history/search, playback/export, and agent mode for controlling a Mac with voice commands.
- Test criteria: speed vs macOS dictation, accuracy for technical terms, Ukrainian/English mixed-language behavior, app compatibility, shortcut ergonomics, privacy settings, and whether the free local model is enough.
