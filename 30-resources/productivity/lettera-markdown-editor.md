# Lettera Markdown Editor

Lettera is a native Markdown editor for macOS from Shiny Frog, the developers of Bear. It started as Panda, the editor built for Bear 2, and is now available as a standalone Mac app in beta through TestFlight.

## Why It Matters

- Focused Mac Markdown editor from a team with a strong writing-app track record.
- Opens either individual Markdown files or whole folders as workspaces.
- Supports WYSIWYG Markdown with CommonMark live rendering, hiding syntax when not editing.
- Includes rich Markdown-adjacent features: tables, quotes, lists, code blocks, inline images and attachments, MathJax formulas, tabs, file/folder management, and a table of contents.
- Can copy or export writing as plain text, rich text, Markdown, HTML, PDF, JPG, ePub, and more.
