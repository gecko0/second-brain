# Chunkr

Source: https://github.com/lumina-ai-inc/chunkr
Captured: 2026-06-28

Chunkr is an open-source document intelligence API from Lumina AI for turning complex documents into RAG/LLM-ready data.

It focuses on:

- Document layout analysis
- OCR with bounding boxes
- Semantic chunking
- Structured HTML and Markdown output
- Vision-language model processing

Supported input types in the open-source version include PDFs, PowerPoint files, Word documents, and images. The README distinguishes the open-source AGPL release from the managed Chunkr Cloud API, which uses proprietary in-house models and adds production-oriented capabilities such as Excel support.

Potential use: evaluate as a self-hosted pipeline for parsing and chunking complex documents before retrieval-augmented generation.
