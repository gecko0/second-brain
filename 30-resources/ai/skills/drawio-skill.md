# drawio-skill

## Summary

`drawio-skill` is an agent skill for generating editable draw.io diagrams from natural-language descriptions, with exports to PNG, SVG, PDF, and JPG.

## Key Points

- [Agents365-ai/drawio-skill](https://github.com/Agents365-ai/drawio-skill) turns natural-language diagram requests into `.drawio` XML and exported image/document formats through the draw.io desktop CLI.
- It supports diagram presets for ERD, UML class, sequence, architecture, ML/deep learning, and flowchart diagrams.
- It can generate codebase structure diagrams for Python, JavaScript/TypeScript, Go, and Rust projects using bundled extractors and auto-layout.
- Useful features include official draw.io shape search, AI/LLM brand logos, visual self-checking, overlap and label fixes, iterative refinement, and reusable style presets.
- It is relevant when an agent needs to produce polished architecture diagrams, system diagrams, UML, ERDs, flowcharts, or editable diagrams for docs and presentations.
- Main dependency to remember: diagram export uses the native draw.io desktop CLI.
