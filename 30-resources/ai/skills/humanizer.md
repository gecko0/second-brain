# Humanizer

Source: https://github.com/blader/humanizer
Captured: 2026-06-28

Humanizer is a Claude Code and OpenCode skill for rewriting text to reduce common signs of AI-generated writing and make the result sound more natural.

Key points:

- MIT licensed.
- Distributed as a `SKILL.md`-based agent skill.
- Supports Claude Code and OpenCode installation.
- Can be invoked directly with `/humanizer` or by asking the model to humanize text.
- Includes voice calibration from a sample of the user's own writing.
- Uses a checklist of AI-writing patterns and a second audit/rewrite pass.

Potential use: reference when refining AI-assisted prose, especially when a draft sounds generic, over-polished, or visibly model-generated.
