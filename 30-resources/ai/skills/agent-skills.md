# Agent Skills

## Summary

Agent Skills is an open-source collection of reusable AI agent skills by Addy Osmani. It is useful as a reference for how agent capabilities can be packaged, documented, shared, and composed.

## Key Points

- Provides examples of discrete skills that can extend an AI coding or productivity agent.
- Useful for studying skill structure, naming, scope, activation patterns, and documentation conventions.
- Relevant to OpenSana and agentic tooling because it shows how agent behaviors can be modularized instead of embedded only in prompts or application logic.
- Worth reviewing for ideas around skill registries, discoverability, reuse, and community contribution patterns.
