# Caveman Skill

## Summary

Caveman is an AI agent skill by juliusbrussee that makes responses terse while preserving technical substance. It is useful as a reference for communication-mode skills and token-efficient agent interaction.

## Key Points

- The skill is installed from `https://github.com/juliusbrussee/caveman` with the `caveman` skill name.
- Its core behavior is to remove filler, pleasantries, hedging, and unnecessary articles while keeping exact technical terms and code unchanged.
- It defines persistence rules: once activated, the style stays active until explicitly disabled.
- It supports intensity levels such as `lite`, `full`, and `ultra`.
- Relevant to AI skill design because it demonstrates a narrow, durable behavior mode with clear activation, deactivation, and formatting rules.
