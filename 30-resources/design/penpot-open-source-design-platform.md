# Penpot Open Source Design Platform

## Summary

Penpot is an open-source design and prototyping platform for teams that want design-code collaboration, self-hosting, design systems, and ownership of their design infrastructure.

## Key Points

- [penpot/penpot](https://github.com/penpot/penpot) is the main open-source repository for Penpot.
- Penpot can be used in the browser as SaaS or self-hosted, which makes it relevant when vendor lock-in, governance, compliance, or infrastructure ownership matter.
- It works with open web standards such as SVG, CSS, HTML, and JSON, and is designed to make design artifacts more readable and actionable for developers.
- Useful product surfaces include real-time collaboration, inspect mode with SVG/CSS/HTML code, design tokens, components, variants, plugins, API access, webhooks, and an MCP server.
- Worth evaluating as a Figma alternative for product teams that want design systems, developer-friendly handoff, automation, AI/code workflows, and open-source deployment options.
