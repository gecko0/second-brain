# Icon Libraries

## Summary

Reference sources for icons and brand/logo SVGs used in design and frontend work.

## Key Points

- [ICONSVG](https://iconsvg.xyz/) is a quick SVG icon customizer/generator. Use it when a project needs a small one-off interface SVG that can be copied or downloaded without adding a package.
- [Circum Icons](https://circumicons.com/icons) is a curated open-source icon library with designer/developer modes, size and color controls, and download/URL options.
- [Heroicons](https://heroicons.com/) is the official hand-crafted SVG icon set by the makers of Tailwind CSS. It has outline, solid, mini, and micro styles, React and Vue packages, and an MIT license.
- [Phosphor Icons](https://phosphoricons.com/) is a general-purpose icon family created by Helena Zhang and Tobias Fried. Use it when a product UI needs consistent interface icons.
- [SVGL](https://svgl.app/) is a searchable SVG logo library. Use it when a project needs product, company, framework, or tool logos.
- [Tabler Icons](https://tabler.io/icons) is a large free/open-source SVG icon set with 6100+ icons, consistent 24x24 grid / 2px stroke styling, multiple formats, React support, and a Figma plugin.
- Keep usage rights and brand guidelines in mind before shipping downloaded assets in client or public work.
